## Getting Started

To get started with this project, follow these steps:

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/asif-daffodil/softwrd.ai.git
   ```

2. **Change Directory:**

   ```bash
   cd softwrd.ai
   ```

3. **Install Dependencies:**

   ```bash
   npm install
   ```

4. **Start the Development Server:**

   ```bash
   npm run dev
   ```

5. **Open in Browser:**

   Open your web browser and navigate to [http://localhost:3000](http://localhost:3000) to see the application in action.

## Features

- **Home Page**: The landing page of the application.
- **Layout Page**: An example of the application's layout.
- **Chart Page**: Displays interactive charts using Chart.js.
- **Calendar Page**: Shows a calendar using the OpenLayers library.

## Built With

- [Svelte](https://svelte.dev/): The web framework used for building the user interface.
- [Vite](https://vitejs.dev/): The build tool and development server.
- [Tailwind CSS](https://tailwindcss.com/): A utility-first CSS framework.
- [Chart.js](https://www.chartjs.org/): For creating interactive charts.
- [OpenLayers](https://openlayers.org/): Used for displaying maps and calendars.

## Folder Structure

- `static/`: Static assets like images, fonts, and other files.
- `public/`: Publicly accessible files.
- `build/`: The build output directory.
